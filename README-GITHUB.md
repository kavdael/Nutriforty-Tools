# NutriForty Tools — Blog-Shop 40+

Web estática lista para GitHub Pages.

## Cómo subir a GitHub

1. Descomprime el ZIP.
2. En GitHub entra en el repo `kavdael/Nutriforty-fitcorner`.
3. Pulsa **Add file → Upload files**.
4. Sube todo el contenido de esta carpeta, incluyendo:
   - `index.html`
   - `tienda.html`
   - `producto.html`
   - `assets/`
   - `data/`
5. Pulsa **Commit changes**.
6. GitHub Pages debe estar en **Settings → Pages → Deploy from a branch → main / root**.

## Cambiar el tag de Amazon

Busca en todo el proyecto:

```txt
TU-TAG-21
```

Y sustitúyelo por tu tracking ID real de Amazon Afiliados.

## Importante

- No subas claves privadas de Amazon al frontend.
- No copies imágenes de Amazon manualmente si no tienes permiso.
- Los precios y disponibilidad deben confirmarse en la tienda de destino o integrarse mediante herramientas/API autorizadas.
- Esta web no hace promesas médicas; es una guía editorial de herramientas para hábitos.
